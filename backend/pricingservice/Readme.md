# 🧠 CartCraft Pricing Service

The Pricing Service is an AI-assisted, rule-based dynamic pricing engine designed for the CartCraft microservices architecture.

It evaluates product variants using real-time behavioral metrics and proposes bounded price adjustments through a deterministic pricing engine validated by an AI reasoning layer.

---

## 🚀 Core Responsibilities

- Monitor variant activity (cart events, sold events)
- Evaluate demand patterns over rolling 24-hour windows
- Generate bounded deterministic pricing adjustments
- Validate pricing strategies using LangGraph AI workflow
- Store pricing proposals for manual approval
- Maintain historical price change records
- Expose Prometheus metrics for observability

---

## 🏗 Architecture Overview

Redis (24h Activity Metrics)
        ↓
Evaluation Scheduler
        ↓
Deterministic Pricing Engine
        ↓
LangGraph AI Validation
        ↓
Pricing Proposal (DB)
        ↓
Admin Approval API
        ↓
Sale Price Update

---

## 📊 Data Sources

- Redis Sorted Sets (cart events & sold events)
- Product Service (base price & inventory context)
- Historical pricing memory stored in PostgreSQL

---

## 🧠 AI Workflow


- Stage 1: Market state reasoning
- Stage 2: Strategy validation
- Memory guard checks
- Interrupt-based approval system

AI does **not directly modify prices**.  
It validates deterministic proposals generated by the pricing engine.

## 🔄 Evaluation Flow

1. Scheduler starts evaluation batch
2. Active variants fetched from Redis
3. 24h activity metrics aggregated
4. Deterministic engine computes bounded adjustment
5. AI validates proposed strategy
6. Proposal stored with `PENDING` status
7. Admin approves or rejects proposal
8. Sale price updated if approved

---

## 🔐 Safety Constraints

- Price changes are strictly bounded
- AI cannot directly change prices
- Manual approval required before applying changes
- Full audit logs maintained
- Evaluation batches tracked for monitoring

## 🐳 Running Locally

### 1️⃣ Install Dependencies

pip install -r requirements.txt

### 2️⃣ Set Environment Variables

DATABASE_URL=postgresql://postgres:postgres@postgres:5432/pricing_db  
REDIS_URL=redis://redis:6379  
OPENAI_API_KEY=your_key  
LANGSMITH_TRACING=true  
LANGSMITH_PROJECT=cartcraft  

### 3️⃣ Run Service

uvicorn main:app --host 0.0.0.0 --port 8000
